**Тестовое задание. Необходимо было: **
- Написать погодное iOS-приложение на Swift
- UIKit, программная верстка
- Отобразить один экран с погодной информацией: текущая, почасовая (показывать оставшиеся часы из текущего дня и все часы следующего), прогноз погоды на 3 дня.
- Обработать показ загрузки и ошибку, если что-то пошло не так, с кнопкой повторного запроса
- Дизайн произвольный
- Запрашивать разрешение на использование геопозиции пользователя при запуске приложения
- Данные получать из заранее выделенного API ключа + обравботать 2 запроса (current и forecast отдельно)

*Стек*
- Архитектура MVP + Coordinator
- Async/Await для многопоточности (требование вакансии)
- Moya для работы с сетью
- CoreLocation для работы с геолокацией
- Snapkit для верстки (верстка преимущественно сделана с использованием UIStackView)
